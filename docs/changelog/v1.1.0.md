# [1.1.0] - 2026-01-19

## Added

- Introduced a **Wallet system** for users:
  - Display real-time user balance.
  - Added a redesigned **payment interface** for a clearer and smoother checkout experience.
- Added **account status filtering** with three states:
  - Available
  - Reserved (Deposited)
  - Sold
- Updated **Order UI for users** to improve clarity and usability during the purchasing process.

## Changed

- Improved **Admin UI/UX**:
  - Enhanced usability and layout for:
    - Account management page
    - Order management page
  - Optimized information hierarchy for faster admin operations.
- Updated image rendering logic:
  - Handle cases where `<img src>` is empty or invalid.
  - Automatically fallback to a **placeholder image** to prevent broken UI.

## Fixed

- Fixed UI issues caused by missing or empty image sources.
- Improved overall UI consistency across admin and user-facing pages.

## UI / UX Improvements

- Refined spacing, alignment, and visual hierarchy in admin dashboards.
- Improved user order flow with clearer status indication and action feedback.
