# [1.2.0] - 2026-02-21

## Added

- Added **Loading Skeletons** across the application:
  - Improve perceived performance during data fetching.
  - Provide smoother visual feedback while loading pages and components.
- Completed and enhanced the **payment system**:
  - Finalized the end-to-end payment flow.
  - Added mechanisms to ensure **transaction integrity** across all related operations.

## Changed

- Improved **Admin Account UI/UX**:
  - Refined layout and visual hierarchy on the admin account management page.
  - Enhanced usability for faster account review and management.
- Migrated **media storage from AWS S3 to Cloudflare R2**:
  - Reduced bandwidth and egress costs.
  - Improved cost efficiency for image and video delivery.
  - Optimized integration with CDN-based asset delivery.
- Polished overall **UI and UX consistency**:
  - Improved spacing, alignment, and component behavior across admin interfaces.
  - Refined interaction feedback for smoother workflows.

## Fixed

- Ensured **transaction consistency and safety**:
  - Prevented duplicate or conflicting transactions.
  - Guaranteed atomic operations between wallet balance, orders, and payment records.
- Fixed UI issues related to loading states and asynchronous data handling.

## UI / UX Improvements

- Introduced skeleton loading states to reduce perceived wait times.
- Improved admin workflows with clearer visual cues and interaction feedback.
- Enhanced reliability and user trust during payment and transaction processes.
